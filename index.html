Políticas de Privacidad
Información General

Este sitio web es un proyecto educativo desarrollado con fines académicos. La página presenta contenido relacionado con videojuegos, específicamente Brawlhalla y Call of Duty: Warzone, con el objetivo de practicar y demostrar habilidades en desarrollo web utilizando HTML, CSS, Bootstrap y JavaScript.

Privacidad del Usuario

Este sitio web no recopila, almacena ni procesa información personal de los usuarios. No existen sistemas de registro, formularios de almacenamiento de datos ni herramientas de seguimiento.

Enlaces Externos

La página puede contener enlaces a plataformas externas como TikTok, Instagram y Twitch. Estos enlaces se proporcionan únicamente con fines informativos y para compartir contenido relacionado con la página.

Aceptación de las Políticas

Al utilizar este sitio web, el usuario acepta estas políticas de privacidad y reconoce que el contenido presentado tiene fines educativos e informativos.
